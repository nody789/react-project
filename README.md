## 簡介
以 React.js 框架實作網站，功能區分前、後台：
* 前台
  * 首頁：圖片輪播、網站導覽
  * Q&A：常見問答頁面
  * 產品頁：瀏覽商品分類，商品詳細內容
  * 搜尋欄：輸入關鍵字，搜尋特定商品
  * 購物車：建立訂單付款流程
  * 收藏:收藏商品
  * 會員中心：瀏覽訂單資料
* 後台
  * 管理者登入：帳號及密碼驗證
  * 產品管理
  * 訂單管理
  * 優惠券管理
  * 文章管理
