## 簡介
以 React.js 框架實作網站，功能區分前、後台：
* 前台
  * 首頁：圖片輪播、網站導覽
  * Q&A：常見問答頁面
  * 產品頁：瀏覽商品分類，商品詳細內容
  * 搜尋欄：輸入關鍵字，搜尋特定商品
  * 購物車：建立訂單付款流程
  * 收藏:收藏商品
  * 會員中心：瀏覽訂單資料
* 後台
  * 管理者登入：帳號及密碼驗證
  * 產品管理
  * 訂單管理
  * 優惠券管理
  * 文章管理

## 安裝流程
1. 開啟終端機，執行以下指令：
 ```
 $ git clone https://github.com/nody789/react-project.git
 ```
2. 進入專案資料夾:
```
 $ cd react-project
```
3. 使用npm安裝套件
```
$ npm install
```
4. 新增.env檔案，設定環境變數
```
REACT_APP_API_URL=https://ec-course-api.hexschool.io/
REACT_APP_API_PATH=nody789-hexschool
```
5. 執行專案
```
$ npm start
```
